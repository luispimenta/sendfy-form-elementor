Send notifications to Sendfy when there is a new WooCommerce order or status change

== Description ==

The event will be send on integration **When there is a new WooCommerce order or status change**.

== Installation ==

1. You need to install and activate WooCommerce first.
2. Then upload **Sendfy Form Elementor** plugin to your blog's `wp-content/plugins/` directory and activate.
3. Config your Whatsapp Instace Key.
